开发者偏爱Linux的9大理由

我们日常使用的大多数设备运行的都是 Linux 或定制版本的 Linux。包括 Android 手机、平板电脑、相机、录像机、可穿戴设备、Chromebook 和其他设备。

有趣的事实：您使用的大多数互联网服务和社交媒体网站都在 Linux 上运行，因为这是最值得信赖的操作系统。

01强大的命令行

　　命令行有很多功能，它使您能够快速开发并自动执行日常任务。很多开发者喜欢它，正是因为它不需要鼠标或触控板。

　　除此以外还能使日常任务可以自动化。这可以帮助开发者专注于手头更重要的任务，并节省大量时间。Linux虽然不完美，但保持了终端的纯度。

02Linux 非常安全

　　因为 Linux 是开源的，由大型开发人员社区开发和维护的，所以捕获和修复安全漏洞的几率更高。此外，Windows 仍然是最受欢迎的操作系统，约占台式机和笔记本电脑市场的76.7%。因此，大多数攻击针对的都是Windows 而不是 Linux。

03对开发人员非常友好

　　与 Windows 用户相比，Linux 比以往任何时候都更加对用户友好。但正因为Linux 专门为开发人员提供的工具，才使其在开发人员中如此受欢迎。

　　开发者会收到一个编译器、命令行编辑器、几乎可以调整任何东西，以及设置你自己的快捷方式。这一切都是开箱即用（默认对所有用户可用，不需支付额外费用或进行另外的配置）的。

04定时更新

　　由于 Linux 是开源的，当人们发现错误、安全缓存和错误脚本时，开发者会定期获得大量更新。这是 Linux 用户的主要优势之一。

　　然而，当 Microsoft 收到一组问题或必须解决任何重大问题时，它会发布软件更新。因此，更新可能会导致您的系统崩溃，或者您可能会发现自己等待更新来解决问题的时间很长。

05有用的社区支持

　　Linux 社区致力于所有 Linux 用户，因此它将提供长期支持，这与 Windows 社区不同，后者刚刚停止为 Windows7客户提供支持，因此如果您有任何问题，您将无法联系帮助中心或解决 Windows7中的数据泄露的问题。

　　您可以在任何地方寻求帮助，例如，在 Twitter、LinkedIn、论坛或重要的 Stack Overflow 上。

06更少的功耗

　　Linux 从计算机中消耗的资源非常少。如果您有一台非常旧的计算机，由于其规格极低而无法正常工作，您仍然可以在该设备上安装 Linux，它应该可以正常工作。

　　众所周知，基于 Windows 的系统需要更强大的硬件。对于当前的 Windows 系统，您的旧硬件支持的 PC 可能会随着操作系统的发展而过时，因为它将不再支持您的旧硬件。

07隐私

　　Linux 不会收集或与任何人共享您的信息。开发者甚至可以关闭用于开发建议的数据收集，如果打开，它也只会发送给 Linux 的开发团队。如果您曾经使用过 Windows 10，您会注意到默认隐私设置启用了所有功能。即使您选择不提交有关您自己的 Microsoft 数据，它仍然会被收集，除非使用一组程序来禁用 Windows 的窃听模块。

08完全免费使用

　　Linux 已经发展成为世界上最可靠的计算机生态系统之一。当将这种可靠性与免费使用的事实结合起来时，您就拥有了理想的桌面平台。

　　如果你想使用Windows，你要么必须购买它，要么像小偷一样使用破解版，但Linux不需要你这么做。

09网络安全经验

　　Linux 通常是网络安全最常用的操作系统，这是因为Linux不容易感染任何病毒，而且它的系统维护得很好，系统中出现错误的可能性非常小。

　　因此，通过使用 Linux，开发者将获得使用它的经验，因此可以学习如何保护您的个人或专业数据免受暴力攻击、网络攻击或任何其他类型的攻击，这些攻击很容易将您的所有数据泄露给黑客。

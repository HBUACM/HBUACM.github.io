每日分享——GitHub

Github https://github.com/
GitHub 是一个面向开源及私有 软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub，其社交化编码的理念伴随着开源运动改变着整个开发社区的生态，无数优质项目依托GitHub在全球开源开发者的参与下蓬勃发展。

GitHub 于 2008 年 4 月 10 日正式上线，除了 Git 代码仓库托管及基本的 Web 管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。目前，其注册用户已经超过350万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。 2018年6月，GitHub被微软以75亿美元的价格收购。

GitHub用户可以在GitHub上创建自己的仓库，将本地的文件、项目等上传至仓库，可以永久存储，并且可以设置仓库为私有或公有，私有状态下只有自己可以访问，同时可以利用github pages创建自己的网站，用于博客等内容。用户还可以加入其他人创办的github组织，多个人共同参与仓库的管理，适用于多人协作，体现分工合作，提高效率。
